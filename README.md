# Clavier d'Or

Application de quiz compétitif avec progression par étapes, rôles et jokers, sauvegarde des scores, export PDF et modèle de données orienté objet, désormais implémentée en C# avec WinForms et Entity Framework Core.

## Lancer l'application

```bash
dotnet restore src/ClavierDor/ClavierDor.csproj
dotnet build src/ClavierDor/ClavierDor.csproj
dotnet run --project src/ClavierDor/ClavierDor.csproj
```
